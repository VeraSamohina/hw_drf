# Платформа онлайн-курсов (урок 27.2)

## Работа с приложением

- Установить Docker, следуя инструкциям для вашей операционной системы.
- УстановитьDocker Compose.
- Открыть терминал в директории с файлом docker-compose.yаml и выполнить следующую команду:

    ``` docker-compose up ```

- После этого Docker Compose: 
  - загрузит и соберет все необходимые образы
  - запустит redis
  - запустит postgres, создаст базу данных и выполнит все необходимые миграции
  - запустит сервер на порту 0.0.0.0:8000
  - запустит celery worker
  - запустит celery beat

- Проверьте приложение:
После успешного запуска, ваше приложение LMS должно быть доступно по адресу http://localhost:8000
